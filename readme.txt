== Description ==

Custom Product List Table plugin is useful to implement product list in wordpress backend.

It provides additional functionality for product's various status like pending/approved/cancel.

Add product with selection of different category and availability to client or distributor.

== Installation ==
1. Upload plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After activation, you can go through wordpress dashboard / custom Product List Table 

   
== Frequently Asked Questions ==

How can I use this plugin?

you can go through dashboard option custom Product List Table and add products with category selection.
